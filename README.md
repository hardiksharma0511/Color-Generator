# 🎨 Random Color Generator

A simple JavaScript practice project that generates a random background color with a single click.

## Features

- Generate random HEX colors
- Change the background color instantly
- Display the HEX color code
- Copy the color code to the clipboard automatically

## Built With

- HTML
- CSS
- JavaScript

## Project Structure

```
Random-Color-Generator/
│── index.html
│── css/
│   └── custom.css
│── js/
│   └── app.js
│── README.md
```

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/hardiksharma0511/Color-Generator.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

## Purpose

This project was built to practice JavaScript concepts such as:
- Functions
- DOM Manipulation
- Event Listeners
- Random Number Generation
- Clipboard API
